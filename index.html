Js ONLY Method

If you are only using javascript, this method will be easier to setup.

./manifest.json

{
"manifest_version": 3,
"name": "your custom extension name",
"description": "Your custom extension description",
"version": "0.0.1",has to follow number.number.number.number pattern (you can user 1-4 numbers)
"content_scripts": [
{
"matches": [
"URLs you want your extension to work on",
"(You can have as many as you want.)"
],
"js": [
"script_path/scriptName.js"
]
}
],
"icons": {
"64": "icon.png"
}
}

Change script_path/scriptName.js to the path to your js file (you can have as many of these as you want). And you can also change the icon.png to any filename and the 64 to the size of the image.

Now if you have an extension, that you can use or send to your friends to use.

Using the extension

(This is the same for most browsers)

  1. Go to chrome://extensions or click Manage Extensions
  2. Click Load unpacked (You will need to turn on developer mode)
  3. Select the folder with the manifest.json file (along with all your other files/folders)

You now have loaded your custom extension.
